# Rust

Rust learning project with a simple CLI todo app.

I believe todo apps on any programing language is one of the fastest way to learn. So built this to learn basics of rust.

## Run

```bash
cargo run -- <command>
```

## Todo commands

| Command | Example |
|--------|--------|
| `list` | `cargo run -- list` |
| `add "task"` | `cargo run -- add "Learn Rust"` |
| `done <number>` | `cargo run -- done 1` |
| `help` | `cargo run -- help` |

Todos are saved in `.todos` in the project folder.
